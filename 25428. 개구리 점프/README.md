# 25428. 개구리 점프
## 문제
길이가 N인 물웅덩이에 개구리가 살고있다. 이 개구리는 물에서 헤엄을 못쳐서, 물 위에 나뭇잎이 떠 있는 곳으로 점프를 해야만 움직일 수 있다.
개구리가 점프할 수 있는 거리는 최대 K이고, 뛸 수 있는 나뭇잎이 없더라도, 개구리는 무조건 K만큼 점프를 하고 물에 빠지게 된다.

물웅덩이에 대한 정보가 주어지고, 물웅덩이의 가장 왼쪽에는 항상 나뭇잎이 떠있다. 개구리는 물웅덩이의 가장 왼쪽에서 점프를 시작한다.
개구리가 최대로 이동한 거리가 얼마인지 알아내는 프로그램을 작성하시오.

나뭇잎이 있는 곳은 1, 없는 곳은 0으로 표시된 길이가 N인 물웅덩이의 정보와 개구리가 한번에 최대로 점프할 수 있는 거리 K가 주어진다.
물웅덩이의 가장 왼쪽에는 항상 나뭇잎이 떠있다. 개구리는 항상 물웅덩이의 가장 왼쪽에서부터 점프를 시작한다.

1 0 0 0 1 0 0 0 1 0 0 0

물웅덩이가 위 처럼 되어 있는 경우, 물웅덩이의 길이는 12 이고, 개구리가 한번에 점프할 수 있는 거리가 4 라고 할때,
개구리는 4칸씩 점프해서 최대 12 까지 점프한 뒤에 물에 빠지게 되고, 최대 이동한 거리는 12가 된다.

1 0 0 1 0 0 1 0

물웅덩이가 위 처럼 되어 있는 경우, 물웅덩이의 길이는 8 이고, 개구리가 한번에 점프할 수 있는 거리가 3 이라고 할때,
개구리는 3칸씩 점프하고, 7번 위치에서 점프를 했을때 원래는 10번 까지 갈 수 있지만, 물웅덩이의 범위를 벗어날 수는 없고 8번 위치에서 멈추게 된다.
따라서 최대 이동 거리는 8이다.

1 1 0 0 0 0 0 0

물웅덩이가 위 처럼 되어 있는 경우, 물웅덩이의 길이는 8이고, 개구리가 한번에 점프할 수 있는 거리가 3 이라고 할때,
개구리는 처음에 1번 위치에서 점프해서 나뭇잎이 있는 2번 위치로 점프하고, 2번 위치에서 갈 수 있는 나뭇잎이 더이상 없으므로 최대 3칸 점프한
5번 위치까지 갈 수 있다.

## [입력]
첫 줄에 물웅덩이의 개수 T가 주어지고, 물웅덩이 별로 첫줄에 물웅덩이의 길이 N과 개구리가 최대로 점프할 수 있는 거리 K가 주어진다.
그 다음줄에 길이가 N인 물웅덩이의 정보가 빈칸으로 구분되어 주어진다.

## [출력]
각 물웅덩이별로 한줄에 #과 1번 부터인 물웅덩이번호, 빈칸에 이어 최대로 점프해서 이동한 거리를 출력한다.
