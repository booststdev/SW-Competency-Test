# SW-Competency-Test

## 개요

이 리포지토리는 **SW Competency Test**를 준비하면서 풀었던 문제들의 풀이를 정리한 공간입니다. 알고리즘, 데이터 구조, 소프트웨어 엔지니어링 등의 분야에서 풀었던 문제들을 기록하고, 그에 대한 해설과 함께 풀이 과정을 공유합니다. 이 리포지토리는 **IM**을 준비하는 데 풀었던 문제를 올립니다.

## 목표

* 다양한 **SW Competency Test** 문제 풀이를 공유
* 각 문제의 풀이 과정과 해설을 통해 문제 해결 방법론을 이해하고 익히기

## 내용

이 리포지토리에는 다음과 같은 내용이 포함되어 있습니다:

* **알고리즘 문제 풀이**: 정렬, 탐색, 동적 프로그래밍 등 다양한 알고리즘 문제
* **데이터 구조 풀이**: 배열, 연결 리스트, 스택, 큐, 트리, 그래프 등

## 사용 방법

1. 이 리포지토리를 클론하여 본인의 문제 풀이를 업로드합니다:

   ```bash
   git clone https://github.com/yourusername/SW-Competency-Test.git
   ```

2. 각 문제를 해결한 후, **문제 번호**를 기준으로 폴더를 생성합니다.

3. 문제에 대한 풀이와 함께 해당 문제를 푼 과정을 **README.md** 파일에 작성하여 업로드합니다.

4. 풀이가 완료되면, 해당 폴더를 커밋하고 푸시하여 최신 상태를 유지합니다.
