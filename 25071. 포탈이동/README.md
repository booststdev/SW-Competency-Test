# 25071. 포탈이동
## 문제
차원 여행자 스트레인지는 이상한 공간에 갇혀버렸다. 이 공간을 탐색한 결과 이 공간은 N개의 방으로 구성되어있고, 각 방에는 1부터 N까지의 번호가 붙어있다.
각 방에는 포탈이 존재하는데 포탈에는 그 방의 번호 i 보다 작은 숫자 Pi가 적혀있다(Pi < i). 포탈을 사용하면 다른 방으로 이동할 수 있는데, 특별한 규칙이 존재한다는 것을 알게되었고, 규칙은 아래와 같다.

각 i번 방의 포탈에는 번호 Pi가 쓰여져 있다.
만약 i번 방에 처음 방문하는 것이라면, 이 i번 방에 있는 포탈을 사용하면 포탈에 적혀있는 번호 Pi 번 방으로 이동하게 된다.
만약 i번 방에 처음 방문하는 것이 아니라면, 이 i번 방에 있는 포탈을 사용하면 바로 한 칸 뒤에 있는 i+1 번 방으로 이동하게 된다.

1번 방의 포탈을 사용하면 방문 여부에 상관없이 2번 방으로 이동한다.
마지막 N번 방에 이 공간의 출구가 존재한다. (출구는 포탈이 아니다.)

1번방에서 포탈을 몇번 사용해야 이 공간에서 나갈 수 있는지 구하는 프로그램을 작성하시오.

**예시)**
N = 5 이고, 각 방의 포탈 번호가 아래와 같을때, 처음 방 번호는 1 이고, 마지막 방 번호는 5번이다.
0 1 1 2 0
포탈을 사용하여 이동하는 방의 순서를 기록하면
1 => 2 => 1 => 2 => 3 => 1 => 2 => 3 => 4 => 2 => 3 => 4 => 5
이므로 탈출하기 위해 포탈을 사용하는 횟수는 12 가 된다.

**제약사항**
i번 방에 있는 포탈을 사용하면 이동하는 방 번호 Pi 는 i보다 작다. (i < Pi)
각 방의 개수 N 의 범위 : 4 <= N <= 20

**입력**
첫 번째 줄에 테스트케이스의 개수 T가 주어진다.
두 번째 줄부터 각 테스트케이스의 입력이 2줄씩 주어진다.
각 테스트 케이스의 첫 번째 줄에는 방의 개수 N 이 주어진다.
각 테스트 케이스의 두 번째 줄에는 각 방에 처음 도착시 포탈을 사용하면 이동하게 되는 방 번호 P1, P2, P3, ... , Pn 이 순서대로 주어진다. 

**출력**
출력 예시 참고

## [입력]
3
5
0 1 1 2 0
5
0 1 1 1 0
4
0 1 2 0

## [출력]
#1 12
#2 13
#3 7
